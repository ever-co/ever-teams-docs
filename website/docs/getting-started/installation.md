# Install Ever Teams Platform

Track the progress of your teams' work in real time

## Installation

To install ever-demand, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/ever-co/ever-teams.git
    ```

2. Navigate to the project directory:

    ```bash
    cd ever-teams
    ```

3. Install dependencies:

    ```bash
    yarn install
    ```

4. Start the development server:

    ```bash
    yarn start:web:dev
    ```

Visit [https://ever.team](https://ever.team) for more detailed installation instructions and troubleshooting tips.
