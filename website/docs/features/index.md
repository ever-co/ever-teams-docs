# Features

Ever Teams provides a rich set of features for work management, time tracking, and team collaboration. Explore each feature in detail:

## Core Features

- **[Time Tracking & Timer](./time-tracking)** — Real-time timer, activity monitoring, time logs, and time limits.
- **[Task Management](./task-management)** — Create, assign, prioritize, and track tasks with custom workflows.
- **[Teams & Organizations](./teams)** — Manage organizations, teams, and member roles.
- **[Projects](./projects)** — Organize work into projects with dedicated views.
- **[Daily Plans](./daily-plans)** — Plan your workday with structured daily task lists.

## Views & Visualization

- **[Kanban Board](./kanban)** — Visual drag-and-drop task board for agile workflows.
- **[Calendar View](./calendar)** — Timeline-based task and schedule visualization.
- **[Dashboard & Analytics](./dashboard)** — At-a-glance metrics and team performance overview.

## Collaboration

- **[Video Conferencing (Meet)](./meet)** — Built-in video meetings with Jitsi or LiveKit.
- **[Timesheet Management](./timesheets)** — Submit, review, and approve team timesheets.
- **[Reports & Analytics](./reports)** — Detailed activity, time, and productivity reports.

## Configuration & Integrations

- **[Integrations](./integrations)** — GitHub, JIRA, and other third-party integrations.
- **[Roles & Permissions](./roles-permissions)** — Role-based access control for team management.
- **[Settings](./settings)** — Application and organization settings.
- **[Internationalization](./i18n)** — Multi-language support across 13+ locales.
- **[Themes](./themes)** — Dark, light, and system theme preferences.
