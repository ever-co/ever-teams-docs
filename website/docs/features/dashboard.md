# Dashboard & Analytics

The Dashboard (`/dashboard`) provides an at-a-glance overview of team activity, productivity metrics, and key performance indicators.

## Overview

The dashboard aggregates data from across the platform to provide:

- **Team activity summary** — Who's working, what they're working on
- **Time tracking metrics** — Hours tracked today, this week, this month
- **Task statistics** — Task counts by status, completions, and trends
- **Productivity insights** — Activity levels and work patterns

## Widgets

### Team Activity

- Real-time view of team members' current status
- Currently tracking / idle indicator
- Current task being worked on
- Today's total tracked time per member

### Time Overview

| Metric         | Description                 |
| -------------- | --------------------------- |
| **Today**      | Hours tracked today         |
| **This Week**  | Cumulative hours this week  |
| **This Month** | Cumulative hours this month |
| **Trend**      | Week-over-week comparison   |

### Task Summary

| Metric          | Description                    |
| --------------- | ------------------------------ |
| **Open**        | Tasks not yet started          |
| **In Progress** | Tasks currently being worked   |
| **Completed**   | Tasks finished today/this week |
| **Overdue**     | Tasks past their due date      |

### Activity Chart

- Line/bar chart showing daily tracked hours
- Activity level percentages over time
- Breakdown by team member

## Customization

The dashboard layout can be customized:

- Rearrange widgets
- Show/hide specific metrics
- Select date ranges
- Choose team or individual views

## Permissions

| Role              | Dashboard Access                 |
| ----------------- | -------------------------------- |
| **Owner/Manager** | Full team dashboard              |
| **Member**        | Personal metrics + team overview |
| **Viewer**        | Read-only team overview          |
