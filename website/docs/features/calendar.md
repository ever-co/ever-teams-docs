# Calendar View

The Calendar View provides a timeline-based visualization of tasks, deadlines, and time entries.

## Overview

Accessible via `/calendar`, this view helps teams:

- **Plan ahead** — See upcoming deadlines at a glance
- **Track schedules** — Visualize workload distribution over time
- **Manage due dates** — Drag tasks to reschedule deadlines

## Views

| View      | Description                              |
| --------- | ---------------------------------------- |
| **Month** | Full monthly calendar grid               |
| **Week**  | 7-day horizontal view                    |
| **Day**   | Detailed single-day view with time slots |

## Features

### Task Display

- Tasks appear on their due dates
- Color-coded by status or priority
- Click to expand task details

### Drag-and-Drop

- Drag tasks to new dates to change due dates
- Drag across time slots in day/week view

### Time Entries

- View tracked time entries on the calendar
- Visualize when team members worked
- Identify gaps or overtime

### Filters

- Filter by team member
- Filter by project
- Filter by priority
- Filter by status
